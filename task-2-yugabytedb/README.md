# Task 2: Data Terdistribusi - YugabyteDB

## Penjelasan Task
Tugas ini fokus pada implementasi database terdistribusi menggunakan **YugabyteDB**.
- **Teori:** YugabyteDB adalah database SQL terdistribusi berperforma tinggi yang kompatibel dengan PostgreSQL. 
- **Implementasi:** Saya mendemonstrasikan cara melakukan deployment cluster YugabyteDB secara lokal, membuat tabel, dan menunjukkan bagaimana replikasi data terjadi di antara node untuk memastikan toleransi kesalahan (*fault tolerance*).
